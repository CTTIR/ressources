Data sources

Public datasets and APIs relevant to multi-omics, biomarker, and trauma / tissue-injury research. Each entry notes the licence terms because they vary widely.

Literature and citation

PubMed · E-utilities

License: PD (most metadata, US-government work) · Maintainer: NCBI / NLM

The biomedical-literature index and its programmatic API. Metadata is free to redistribute; full-text articles follow each publisher’s licence.

Europe PMC

License: varies; metadata under CC0 / CC BY · Maintainer: EMBL-EBI

Mirror and extension of PubMed Central with broader open-access full-text coverage and a documented REST API.

Genomics repositories

NCBI GEO — Gene Expression Omnibus

License: PD (US-government work) · Maintainer: NCBI

The largest functional-genomics archive: arrays, RNA-seq, ChIP-seq, ATAC-seq. Per-submission deposit terms govern reuse; most data are openly available.

ArrayExpress / BioStudies

License: CC0 (most studies) · Maintainer: EMBL-EBI

The European functional-genomics archive, now part of BioStudies. Many studies are dual-deposited with GEO.

Ensembl

License: varies; most annotation under Apache-2.0 / CC0 · Maintainer: EMBL-EBI

Comprehensive genome annotations and a stable REST API. The reference for transcript-level annotation in many R/Bioconductor pipelines.

UCSC Genome Browser

License: varies; track licences declared per dataset · Maintainer: UCSC

Genome visualisation and a large library of annotation tracks. The Table Browser exposes the underlying tables programmatically.

Proteins and pathways

UniProt

License: CC · CC BY 4.0 · Maintainer: UniProt Consortium (EBI / SIB / PIR)

The reference protein-sequence and functional-annotation database. Programmatic access via REST and SPARQL.

Reactome

License: CC · CC BY 4.0 · Maintainer: Reactome Consortium

Curated biological-pathway database with R access via ReactomePA.

Clinical

ClinicalTrials.gov

License: PD (US-government work) · Maintainer: NIH / NLM

Registry of clinical trials worldwide. The API v2 is well documented and returns structured study, eligibility, intervention, and outcome data.

OMIM — Online Mendelian Inheritance in Man

License: registration required for API; site browsable openly; redistribution restricted · Maintainer: Johns Hopkins University

Catalogue of human genes and genetic disorders. The redistribution restrictions are real — read the licence before mirroring data.

ClinVar

License: PD (NCBI submissions) · Maintainer: NCBI

Aggregated reports of relationships between human variants and phenotypes. Used directly by CTIR’s molpathR variant-annotation flow.
